06-Команды find cut sort wc

1) find - ищет файлы
man find - для справки о команде

2) Ищет все файлы по имени
find /home -name "*.txt"
sudo find /home -name "*.txt"

3) wc - показывает кол-во строк в файле, слов, символов в файле
wc data.txt 
wc -l - только кол-во строчек
wc -w - только кол-во слов

4) sort  - сортирует содержимое файла, сам файил не меняется
sort data.txt

5) cut - позволяет вывести на консоль только определенное содержимое файла по строке.
Нужно указать делиметр -d (каждая строка будет парситься)
И номер поля -f (нумерация c 1)

cut -d ">" -f 3 data.txt 
cut -d ">" -f 3,2 data.txt  - выводит несколько полей (с разделителем)

6) В линуксе можно запустить сразу последовательонсть команд (через pipe |)
cut -d ">" -f 3 data.txt  | sort
реузльтат cut еще и сортируется

cut -d ">" -f 3 data.txt  | sort | wc
еще и дает инфо по содержимому
