01-Типы аккаунтов и важные файлы

Существует 2 типа аккаунтов:
a) Standard
b) Administrator - может использовать sudo

1) Место где сохраняются все папки пользователя
/home

cd /home
ls -l  - даст папки с названиями всех пользователей

2) Просмотр инфо о пользователях. 
cat /etc/passwd - выводит длинный текст, разделенный :

petya:x:1001:1001::/home/petya:

petya - имя ползьователя
х - Сам пароль храниться в другом месте
1001 - userid в системе
1001 - groupid (для каждого пользователя создается его группа)
:: - пустота - это полное имя ползьователя (денис прокопюк)
/home/petya - домашняя директория пользователя
: - последний параметр (сейчас пустой) - это default shell (/bin/bash)

3)Просмотр зашифрованых паролей
cat /etc/shadow - ошиибка, только судо
sudo cat /etc/shadow - даже если пароли одинаковые, то зашифрованы они будут по разному.
petya:encr(password)

DBW0:17125:0:99999:7:::
17125 - сколько дней прошло со смены пароля
0 - когда можно поменять пароль
99999 - когда нужно поменять пароль
7 -експайред пароль


4) Список всех групп
cat /etc/group
Выводит
sudo:x:27:denis,kolya

5) Узнать свое имя
whoami

6) Переключиться на другого пользователя 
su vasya
(далее нужно ввести пароль)

7) Просомтреть инфо про себя (userid, group, ...)
id 
id user - выдает инфо про другого ползьователя

8) Кто последним логинелся в систему 
last

9) кто еще паралелльно залогинен в систему
who

10) кто еще паралелльно залогинен в систему, но боллше инфо - кто что сейчас делает
w
